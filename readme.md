# Coding Challenges

This repository contains coding challenges for different experience levels.

## Challenges

### 👶 Junior Level — [`junior/`](./junior/readme.md)

A beginner-friendly challenge focused on building a simple full-stack Svelte application that persists data to a database.

**Key topics:** Svelte, basic CRUD, database models (customers, stores, products)

### 🧑‍💻 Senior Level — [`senior/`](./senior/readme.md)

A more advanced challenge involving ETL pipelines, data cleaning, and full-stack development with real-world grocery data.

**Key topics:** Svelte, ETL, SQL database design, data quality, CSV processing
